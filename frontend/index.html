API Reference

Simple, RESTful endpoints compatible with OpenAI SDK

GET /health

Check server health, model status, and configuration.

curl http://localhost:8000/health
GET /v1/models

List available models (OpenAI-compatible endpoint).

curl http://localhost:8000/v1/models \ -H "Authorization: Bearer local-qwen"
POST /v1/images/generations

Generate images from text prompts. Fully compatible with OpenAI's API.

curl http://localhost:8000/v1/images/generations \ -H "Content-Type: application/json" \ -H "Authorization: Bearer local-qwen" \ -d '{ "prompt": "A beautiful sunset over mountains", "size": "1024x1024", "n": 1 }'

Request Parameters

Parameter Type Default Description
prompt string required Text description of desired image
size string 1024x1024 Image dimensions
n integer 1 Number of images (1-4)
response_format string b64_json b64_json or url
output_format string png png, webp, or jpeg
seed integer random Seed for reproducibility
negative_prompt string "" What to avoid in generation
num_inference_steps integer 20 Number of diffusion steps
true_cfg_scale float 4.0 CFG guidance scale

Python with OpenAI SDK

from openai import OpenAI import base64 from pathlib import Path client = OpenAI( base_url="http://localhost:8000/v1", api_key="local-qwen" ) response = client.images.generate( model="qwen-image", prompt="A fantasy castle in the clouds, golden hour", size="1024x1024", extra_body={ "response_format": "b64_json", "seed": 42, "num_inference_steps": 20, "negative_prompt": "blurry, low quality" } ) # Save the generated image image_data = base64.b64decode(response.data[0].b64_json) Path("castle.png").write_bytes(image_data) print("Image saved to castle.png")

JavaScript / TypeScript

const response = await fetch('http://localhost:8000/v1/images/generations', { method: 'POST', headers: { 'Content-Type': 'application/json', 'Authorization': 'Bearer local-qwen' }, body: JSON.stringify({ prompt: 'A serene lake at sunset, mountains in background', size: '1536x1024', n: 1, response_format: 'b64_json', output_format: 'png' }) }); const data = await response.json(); // Display the image const img = new Image(); img.src = `data:image/png;base64,${data.data[0].b64_json}`; document.body.appendChild(img);

cURL Command

curl -X POST http://localhost:8000/v1/images/generations \ -H "Content-Type: application/json" \ -H "Authorization: Bearer local-qwen" \ -d '{ "prompt": "A futuristic city at night, cyberpunk style", "size": "1536x1024", "n": 1, "response_format": "b64_json", "output_format": "png", "num_inference_steps": 20, "true_cfg_scale": 4.0 }' \ | jq -r '.data[0].b64_json' | base64 -d > image.png
