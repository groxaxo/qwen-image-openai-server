API Reference

Simple, RESTful endpoints compatible with OpenAI SDK

GET /health

Check server health and configuration status.

curl http://localhost:8000/health
GET /v1/models

List available models.

curl http://localhost:8000/v1/models \ -H "Authorization: Bearer local-qwen"
POST /v1/images/generations

Generate images from text prompts.

curl http://localhost:8000/v1/images/generations \ -H "Content-Type: application/json" \ -H "Authorization: Bearer local-qwen" \ -d '{ "prompt": "A beautiful sunset over mountains", "size": "1024x1024", "n": 1 }'

Request Parameters

Parameter Type Default Description
prompt string required Text description of desired image
size string 1024x1024 Image dimensions
n integer 1 Number of images (1-4)
response_format string b64_json b64_json or url
output_format string png png, webp, or jpeg
seed integer random Seed for reproducibility
negative_prompt string "" What to avoid in generation
num_inference_steps integer 20 Number of diffusion steps

Python (OpenAI SDK)

from openai import OpenAI import base64 from pathlib import Path client = OpenAI( base_url="http://localhost:8000/v1", api_key="local-qwen" ) resp = client.images.generate( model="qwen-image", prompt="A fantasy castle in the clouds", size="1024x1024", extra_body={ "response_format": "b64_json", "seed": 42, "num_inference_steps": 20 } ) Path("output.png").write_bytes( base64.b64decode(resp.data[0].b64_json) )

JavaScript (fetch)

const response = await fetch('http://localhost:8000/v1/images/generations', { method: 'POST', headers: { 'Content-Type': 'application/json', 'Authorization': 'Bearer local-qwen' }, body: JSON.stringify({ prompt: 'A serene lake at sunset', size: '1024x1024', n: 1, response_format: 'b64_json' }) }); const data = await response.json(); const img = new Image(); img.src = `data:image/png;base64,${data.data[0].b64_json}`;
